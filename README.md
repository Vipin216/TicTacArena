# TicTacArena

A real-time multiplayer Tic-Tac-Toe platform built with **Django**, **Django Channels**, and **PostgreSQL**. The project focuses on backend engineering concepts such as authentication, WebSockets, matchmaking, real-time synchronization, and persistent game data.

> **Status:** 🚧 In Development

## Tech Stack

### Backend

* Django
* Django Channels *(to be integrated)*
* PostgreSQL

### Frontend

* HTML
* CSS
* JavaScript

### Deployment

* AWS EC2
* Nginx
* Daphne

## Planned Features

* User Registration & Login
* User Profiles
* Room Creation & Joining
* Real-Time Multiplayer using WebSockets
* Matchmaking Queue
* Match History
* Player Statistics
* ELO Rating System
